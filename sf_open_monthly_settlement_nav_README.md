# 顺发打单自动化工具

## 简介

`sf_open_monthly_settlement_nav.py` 是一个 Windows 桌面自动化脚本，用于自动操作"顺发打单"应用程序，实现订单批量导出的自动化操作。

该脚本通过模拟鼠标点击的方式，自动完成以下操作流程：
1. 点击 **"全部订单"** 按钮
2. 点击 **"导出"** 按钮
3. 点击 **"提交"** 按钮

## 功能特点

| 特性 | 说明 |
|------|------|
| **相对坐标定位** | 点击坐标基于窗口相对位置（0-1比例），适应不同屏幕分辨率和窗口尺寸 |
| **录制模式** | 首次运行通过交互式录制按钮位置，简单直观 |
| **循环执行** | 支持定时重复执行，适合需要周期性导出的场景 |
| **自动启动应用** | 如应用未运行，会自动启动桌面快捷方式 |
| **安全保护** | 内置 FAILSAFE 机制，鼠标移至屏幕左上角可紧急终止 |

## 环境要求

- **操作系统**: Windows
- **Python 版本**: Python 3.7+

## 安装依赖

```bash
pip install pywinauto pywin32 pyautogui
```

## 快速开始

### 第一步：首次使用 - 录制按钮位置

首次使用前，需要录制三个按钮的点击位置：

```bash
python sf_open_monthly_settlement_nav.py --record
```

按照提示操作：
1. 确保顺发打单应用已打开
2. 脚本会自动定位并最大化窗口
3. 依次将鼠标移动到各按钮中心，按回车记录：
   - "全部订单" 按钮
   - "导出" 按钮
   - "提交" 按钮

录制完成后，坐标数据会保存到 `sf_open_monthly_settlement_nav_points.json` 文件。

### 第二步：正常执行

```bash
python sf_open_monthly_settlement_nav.py
```

脚本会自动：
1. 查找或启动顺发打单应用
2. 最大化窗口
3. 按顺序执行三个按钮点击
4. 每次点击间隔 1.2 秒

### 第三步（可选）：循环执行

如需定时重复执行：

```bash
# 每 5 分钟执行一次（300秒）
python sf_open_monthly_settlement_nav.py --interval-seconds 300

# 每 10 分钟执行一次（600秒）
python sf_open_monthly_settlement_nav.py --interval-seconds 600
```

按 `Ctrl + C` 可停止循环执行。

## 命令行参数

| 参数 | 说明 |
|------|------|
| `--record` | 录制模式，记录三个按钮的鼠标点击位置 |
| `--interval-seconds <秒数>` | 循环执行模式，按指定秒数间隔重复执行 |

## 配置文件

坐标数据保存在与脚本同目录的 `sf_open_monthly_settlement_nav_points.json`：

```json
{
  "全部订单": {
    "rel_x": 0.1234,
    "rel_y": 0.5678
  },
  "导出": {
    "rel_x": 0.2345,
    "rel_y": 0.6789
  },
  "提交": {
    "rel_x": 0.3456,
    "rel_y": 0.7890
  }
}
```

- `rel_x`: 按钮中心相对于窗口宽度的比例（0-1）
- `rel_y`: 按钮中心相对于窗口高度的比例（0-1）

## 常见问题

### Q: 脚本找不到顺发窗口？
**A:** 检查以下几点：
1. 顺发打单应用是否已启动
2. 窗口标题是否包含"顺发"字样
3. 如未启动，确保快捷方式路径正确（脚本第28行）

### Q: 点击位置不准确？
**A:** 重新执行 `--record` 模式录制坐标：
1. 确保窗口在录制时已最大化
2. 鼠标准确移动到按钮中心
3. 录制过程中不要移动或调整窗口

### Q: 如何紧急停止脚本？
**A:** 有两种方式：
1. 按 `Ctrl + C` 终止程序
2. 快速将鼠标移动到屏幕左上角（FAILSAFE机制）

### Q: 可以修改按钮点击间隔吗？
**A:** 可以，修改脚本第32行的 `CLICK_INTERVAL_SECONDS` 常量。

### Q: 窗口标题不是"顺发"怎么办？
**A:** 修改脚本第29行的 `WINDOW_TITLE_HINT` 为正确的标题关键字。

## 注意事项

1. 执行过程中请勿移动顺发窗口或调整其大小
2. 确保录制时窗口处于最大化状态
3. 建议在顺发打单应用无弹窗、无异常状态下运行
4. 循环执行模式下请确保系统不会进入休眠

## 技术实现

### 依赖库说明

| 库 | 用途 |
|------|------|
| `pywinauto` | Windows 窗口查找、控制和管理 |
| `pyautogui` | 鼠标移动和点击模拟 |
| `pywin32` | Windows API 调用，启动快捷方式 |

### 核心逻辑

```
启动/查找应用
      ↓
最大化窗口
      ↓
读取坐标配置
      ↓
循环执行三个步骤:
  1. 计算绝对坐标（相对坐标 × 窗口尺寸）
  2. 移动鼠标到目标位置
  3. 执行点击
  4. 等待间隔时间
```
